# WallColorMatch

### 간단한 터치로 이루어진 하이퍼캐주얼 게임
게임을 시작하면 플레이어(볼)와 벽에 색상이 나타남. 플레이어(볼)와 동일한 색상의 벽을 맞추면 점수가 올라가고, 스코어에 따라서 레벨이 올라가고 레벨에 따라서 벽 갯수가 많아지고 작아짐. 플레이어가 다른 색상에 부딪히면 게임이 끝나고, 현재 점수와 최고 점수를 확인할 수 있음.
